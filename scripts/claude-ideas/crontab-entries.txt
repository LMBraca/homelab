#   Account 2: 05:00, 09:30, 14:00, 18:30, 23:00   (5 runs)
#   Account 3: 06:00, 10:30, 15:00, 19:30, 00:00   (5 runs)
#
# To install: crontab -e  (then paste the lines below,
# replacing any existing claude-idea-generator / fetch-claude-limits /
# keepalive-claude-tokens / alert-claude-limits lines)
# ============================================================
